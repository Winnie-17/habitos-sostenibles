Reflexión sobre el desarrollo del proyecto

Este proyecto ha permitido aplicar conocimientos técnicos y al mismo tiempo reflexionar sobre el impacto de los hábitos sostenibles. A continuación se presentan las herramientas utilizadas y su propósito en el desarrollo de esta aplicación:

  • Python y Flask: base del sistema para crear el servidor web.
  • HTML y CSS: para estructurar y diseñar visualmente la interfaz.
  • SQLite y SQLAlchemy: para almacenar los datos de hábitos y registros.
  • Render: plataforma gratuita usada para desplegar el sistema online.
  • Git y GitHub: para controlar versiones y publicar actualizaciones.

Además de las herramientas técnicas, la reflexión permitió comprender cómo pequeños hábitos pueden gestionarse digitalmente y cómo una herramienta sencilla puede fomentar la conciencia ambiental entre estudiantes.
